INTRODUCTION
~~~~~~~~~~~~
    RGHF is a very easy to use history utility for Renegade BBS 
    Systems.  RGHF will provide extensive and accurate information
    about a specified date in your BBS's history. 
    
    RGHF is freeware, and registration is not required.  But 
    donations aren't refused. :)


INSTALLATION INSTRUCTIONS
~~~~~~~~~~~~~~~~~~~~~~~~~
    Installation of RGHF is very easy!  The program must be run from your
    main BBS subdirectory where Renegade.Dat is located.  RGHF must find
    this file in the current subdirectory or it will not run.
    
    Now to obtain info on a date, just run the program, and enter a date.
    The date should be in the form of MM-DD-YR, and the BBS had to have 
    been running on that date.  Enjoy!
