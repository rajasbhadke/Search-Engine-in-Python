                                EXPLANATION OF CODES
                                --------------------
MODEM TYPE:  HST uses US Robotics proprietary high speed transfer, V32/V42 are
standard protocols for high speed transfers, DUAL modems can handle both HST 
and v32/v42. (Note, 9600 with v32 code is v32, 14,400 with v32 code is v32.b)

SFTWR ):  The type of software used to run the bbs.  A = Remote Access, 
B = RoboBoard, C = Citadel, E = Renegade, F = FirstClass, G = MajorBBS, 
H = Hermes, M = Maximus, N = Transcendence, O = Opus, P = Spitfire,
Q = QuickBBS, R = RBBS, S = Super BBS, T = Telegard, U = Auntie, W = Wildcat!, 
Y = Harmony, I = TBBS, 1 = C-Net, 2 = ImageBBS, 4 = WWIV, 5 = Tempest, 
6 = Paragon, 7 = TeleFinder, 8 = Excelsior, 9 = Synchronet, V = Virtual,
! = Custom Program.

GAMES=Online Games FILES=Files Section ECHOS=Echo/Net Mail OFFLN=Offline Mail 

SUPPORT/NOTES:   $ = Pay board, $D = donations required for higher access, 
* = must be 16 or older for access, CD = CD/ROM, C = changed since last list, 
N = New this list, G = Gay/lesbian/bi

OTHER NOTES:  Boards listing a phone number which includes commas
are using a device to direct incoming calls.  The commas and the two digit
number following them MUST be used in order to connect to the bbs.
